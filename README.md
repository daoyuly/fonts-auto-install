# fonts-auto-install

## How to install?

### Mac
```shell
curl -L https://github.com/daoyuly/fonts-auto-install/raw/master/install-mac.sh | bash
```

### Linux
```shell
curl -L https://github.com/hbin/top-programming-fonts/raw/master/install-linux.sh | bash
```
